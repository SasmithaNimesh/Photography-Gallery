# SN Photography Portfolio

📸 Personal Note
This project was created during my leisure time to bridge the gap between "logical system building" (my day job) and "artistic expression." It serves as a playground for testing new UI trends like custom cursors and minimalist editorial layouts.

A premium, minimalist photography portfolio concept built for **R.M. Sasmitha Nimesh**. This project was developed as a creative exercise to master advanced CSS layouts, JavaScript-driven interactions, and responsive design principles.

## ✨ Key Features
* **Custom Cursor System:** A dual-element smooth-follow cursor that reacts to interactive elements.
* **Noise Texture Overlay:** A subtle SVG filter for a tactile, "analog" film feel.
* **Dynamic Gallery Filtering:** Category-based filtering (Nature, Macro, Water) with smooth opacity transitions.
* **Fluid Typography:** Uses CSS `clamp()` for headers that scale perfectly across all screen sizes.
* **Responsive Masonry:** A CSS-column-based gallery that adapts from 3 columns (desktop) to 1 column (mobile).
* **Scroll Reveal:** Intersection Observer API used to trigger entrance animations.

## 🛠️ Built With
* **HTML5** - Semantic structure.
* **CSS3** - Custom properties (variables), Grid, Flexbox, and complex animations.
* **JavaScript (ES6+)** - DOM manipulation, Intersection Observer, and custom animation loops.
* **Google Fonts** - *Cormorant Garamond* for elegance and *Montserrat* for modern readability.
